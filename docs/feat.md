# feat.md — 优化路线图

> 基于代码现状整理的优化路线图，按类别分组，标注优先级（P0 高 / P1 中 / P2 低）与实施状态。
> ✅ 已完成 / ❌ 不做 / ⏳ 待办

---

## 一、搜索体验

### 1. 升级为 FlexSearch 全文检索 — P0 ✅ 已完成
- **现状**：[search.js](../iris/js/search.js) 使用 FlexSearch 构建标题 + 正文 + 路径复合索引，自定义 CJK 编码器支持中文分词与模糊匹配。

### 2. 搜索结果关键词高亮 — P1 ✅ 已完成
- **现状**：结果 title/preview 中匹配片段用 `<mark>` 标签包裹高亮。

### 3. 搜索范围切换 — P1 ❌ 已移除
- **决策**：用户反馈「区分标题、正文搜索的功能没什么用」，已删除该功能。

### 4. 搜索路径硬编码修复 — P0 ✅ 已完成
- **现状**：直接使用索引中的完整 `path`，移除了 `'docs/' + path` 硬编码拼接。

---

## 二、阅读体验

### 5. 目录树（Index）可折叠与高亮 — ❌ 不做
- **决策**：保持当前扁平列表 + 左边界高亮现状。

### 6. 阅读进度增强 — ❌ 不做
- **决策**：保持仅顶部一条整体进度条。

### 7. 上一篇 / 下一篇导航 — ✅ 已完成
- **现状**：悬浮球菜单提供「上一篇 / 下一篇」按钮，复用 `fileTree.getAdjacentFiles`。

### 8. 面包屑交互增强 — ❌ 不做
- **决策**：保持现状，仅展示路径。

### 8.5 文件树滚动条隐藏 — ✅ 已完成
- **现状**：文件树展开模式下使用 Shadow DOM 隐藏横向滚动条，保留滚动功能。

### 8.6 长表格左右滚动优化 — ✅ 已完成
- **现状**：长表格自动包裹 `.table-wrapper` 支持横向滚动。

---

## 三、主题与外观

### 9. 跟随系统明暗主题 — ❌ 不做
- **决策**：保持手动选择主题。

### 10. 可视化主题自定义（取色器） — P1 ✅ 已完成
- **现状**：设置面板提供强调色 + 中性色取色器，支持亮色/暗色主题定制，保存到 localStorage。

### 10.5 中性色组扩展 — ✅ 已完成
- **现状**：在原有强调色基础上，新增背景/表面/边框/文字/次要文字 5 个中性色取色器，自由度大幅提升。

### 11. 字体/字号/行高可调 — ❌ 不做
- **决策**：保持固定字体样式（可通过自定义 CSS 自行调整）。

---

## 四、性能与加载

### 12. 首屏性能优化 — P1 ✅ 已完成
- **现状**：关键 CSS/脚本 preload、文件树预构建、路由按需加载文档。

### 13. PWA 离线能力增强 — P1 ✅ 已完成
- **现状**：[sw.js](../sw.js) 预缓存首屏静态资源，文档网络优先 + 离线降级缓存，资源更新时 toast 提示用户刷新。

### 14. 构建脚本增量构建 — ❌ 不做
- **决策**：保持每次全量重建。

---

## 五、内容渲染

### 15. 代码块增强 — P1 ✅ 已完成
- **现状**：每个代码块注入复制按钮、语言标签，`data-lang` 属性便于扩展，长行横向滚动。

### 16. 图片灯箱 — P2 ✅ 已完成
- **现状**：点击图片打开全屏灯箱，支持滚轮缩放、左右键翻页、Esc 关闭，自动收集当前文档所有图片支持画廊模式。

## 六、可访问性与交互

### 17. 可访问性（a11y）系统补齐 — ❌ 不做
- **决策**：暂不处理，后续再说。

### 18. 全局快捷键 — ❌ 不做
- **决策**：仅保留 Esc 关闭侧栏/灯箱。

### 18.5 悬浮球菜单扩展 — ✅ 已完成
- **现状**：从 4 个入口扩展到 7 个：回到顶部、上一篇、下一篇、打开本地 MD、安装 PWA、编辑此页、设置。

### 18.6 打开本地 MD — ✅ 已完成
- **现状**：悬浮菜单提供「打开本地 MD」按钮，使用 FileReader 读取本地文件预览（不写入 URL，刷新后丢失）。

## 七、SEO 与分享

### 19. 动态 SEO meta 与 Open Graph — P2 ✅ 部分完成
- **现状**：每篇文档动态更新 `document.title`（frontmatter.title 优先，其次首个 `# 标题`）。
- **待补全**：OG meta（`og:title`/`og:description`/`og:url`）目前仍是静态的，未来可扩展为动态。

### 20. 标题锚点分享（复制直达链接） — P2 ✅ 已完成
- **现状**：标题悬浮出现「复制链接」按钮，分享直达章节。

## 八、输出与订阅

### 21. 导出 PDF — ✅ 已完成
- **现状**：设置面板 → 操作 → 导出 PDF，通过 `window.print()` + `@media print` 样式隐藏 UI 元素。

### 22. RSS 源 — ✅ 已完成
- **现状**：[build-feed.js](../iris/scripts/build-feed.js) 扫描 docs/ 生成 RSS 2.0 feed.xml，GitHub Actions 自动重建，index.html 添加 `<link rel="alternate">` 支持自动发现。详见 [RSS 订阅](rss.md)。

### 22.5 自定义高亮 JS — ✅ 已完成
- **现状**：设置面板 → 高级 → 自定义高亮 JS，加载外部 highlight.js 主题 CSS URL，回车应用。
