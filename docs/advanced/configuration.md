# 高级配置

学习如何配置你的 Markdown 预览器。

## 基本配置

在 `app.js` 中配置仓库信息：

```javascript
const CONFIG = {
  owner: '你的用户名',
  repo: '你的仓库名'
};
```

## 自动发现说明

本工具通过 GitHub API 自动发现仓库中的所有 `.md` 文件，无需手动配置文件列表！

## 如何添加新文档

1. 在仓库的任意位置创建新的 `.md` 文件
2. 提交并推送到 GitHub
3. 刷新页面，新文档会自动出现在侧边栏

## 自动排序

文件树会自动排序：
- 文件夹排在前面
- 按名称字母顺序排列

## 最佳实践

- 使用有意义的文件名
- 保持文档结构清晰
- 定期更新文档
