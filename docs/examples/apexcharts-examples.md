# ApexCharts 交互式图表示例

本页面展示 ApexCharts 的各种图表类型和配置方式。

---

## 1. 折线图

### 基础折线图

**源码：**
```txt
{
  "chart": {"type": "line", "height": 350},
  "series": [{"name": "销量", "data": [30, 40, 35, 50, 49, 60, 70]}],
  "xaxis": {"categories": ["周一", "周二", "周三", "周四", "周五", "周六", "周日"]}
}
```

**渲染效果：**

```apexcharts
{
  "chart": {"type": "line", "height": 350},
  "series": [{"name": "销量", "data": [30, 40, 35, 50, 49, 60, 70]}],
  "xaxis": {"categories": ["周一", "周二", "周三", "周四", "周五", "周六", "周日"]}
}
```

---

### 多系列折线图

**源码：**
```txt
{
  "chart": {"type": "line", "height": 350},
  "series": [
    {"name": "销量", "data": [30, 40, 35, 50, 49, 60, 70]},
    {"name": "利润", "data": [20, 30, 25, 35, 40, 50, 55]}
  ],
```

## 总结

ApexCharts 提供了丰富的交互式图表类型，包括：

- **折线图/面积图**：展示趋势变化
- **柱状图/条形图**：比较分类数据
- **饼图/环形图**：展示占比关系
- **散点图/气泡图**：展示相关性
- **雷达图**：多维度对比
- **极坐标图**：循环数据展示
- **烛台图**：金融数据展示
- **瀑布图**：财务流水展示

所有图表都支持交互功能，包括缩放、导出、数据点悬停提示等。
