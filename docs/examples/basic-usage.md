# 基础使用教程

学习如何使用 Markdown 预览器的基本功能。

## 界面概览

### 侧边栏
- 自动发现并显示所有 .md 文件的树形结构
- 点击文件夹可以展开/折叠
- 点击文件名加载文档

### 主内容区
- 显示 Markdown 渲染后的内容
- 支持代码高亮
- 支持图片预览

### 进度指示器
- 顶部：文件加载进度条
- 底部：阅读位置进度条

## 基本操作

### 浏览文档

1. 点击侧边栏中的文件名
2. 文档内容将在主区域显示
3. 使用滚动查看完整内容

### 切换文档

- 直接点击其他文件名
- 或使用移动端的汉堡菜单

### 添加新文档

直接在仓库中添加新的 `.md` 文件，提交后刷新页面即可自动发现！

### 阅读进度

滚动页面时，底部的进度条会显示当前阅读位置。

## 代码块交互

点击代码块可以复制内容到剪贴板。

```javascript
console.log('Hello, World!');
```

## 响应式设计

- **桌面端**：侧边栏始终可见
- **移动端**：侧边栏默认隐藏，点击菜单按钮打开

## 快捷操作

- `Esc`：关闭移动端侧边栏
- 点击代码块：复制代码
