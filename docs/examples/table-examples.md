# 表格使用示例

本页面展示各种场景下的表格渲染效果，包括宽表格、长内容表格、多列表格等。

> 💡 提示：表格内容过宽时可左右滑动查看，表头会保持固定可见。

---

## 基础表格

标准的 Markdown 表格语法，支持对齐方式设置。

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容 1 | 内容 2   | 内容 3 |
| 内容 4 | 内容 5   | 内容 6 |
```

**渲染效果：**

| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容 1 | 内容 2   | 内容 3 |
| 内容 4 | 内容 5   | 内容 6 |

## 长内容单元格

单元格内容较长时，会在适当位置自动换行，保持可读性。

### 长文本描述

| 属性 | 说明 |
| ---- | ---- |
| 自适应布局 | 表格支持响应式自适应布局，当内容超出容器宽度时，会自动出现横向滚动条，用户可以通过左右滑动来查看完整的表格内容，同时表头会保持固定位置，方便用户在滚动时对照列名查看对应的数据。 |
| 智能换行 | 单元格内的文本会在单词边界处进行智能换行，不会把一个完整的单词从中间断开，确保阅读体验不受影响。最大宽度限制为 500 像素，超过这个宽度的内容会自动换行显示。 |
| 最小宽度保护 | 每一列都有最小宽度保护，防止列宽被压缩得过窄导致内容无法正常阅读，最小宽度设置为 80 像素。 |

## 混合内容表格

表格中包含各种类型的内容，如代码、链接、强调文本等。

| 功能 | 语法示例 | 说明 |
| ---- | -------- | ---- |
| **粗体** | `**文本**` | 使文本加粗显示，用于强调重要内容 |
| *斜体* | `*文本*` | 使文本倾斜显示，用于标注特殊含义 |
| `代码` | `` `代码` `` | 行内代码样式，用于标识变量名或简短命令 |
| [链接](#) | `[文字](url)` | 超链接，点击跳转到指定地址 |
| ~~删除线~~ | `~~文本~~` | 删除线效果，用于表示已废弃的内容 |

## 特性说明

本项目的表格渲染具有以下特性：

- **横向滚动**：表格宽度超出容器时自动出现横向滚动条
- **智能换行**：在单词边界处换行，保持内容可读性
- **表头固定**：横向滚动时表头保持可见，方便对照查看
- **奇偶行区分**：偶数行有浅背景色，提升阅读体验
- **最小宽度保护**：每列最小宽度 80px，防止过窄
- **最大宽度限制**：单元格最大宽度 500px，超长内容自动换行
